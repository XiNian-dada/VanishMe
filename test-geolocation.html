为什么 Google Maps 显示真实位置？

Google Maps 主要使用 Wi-Fi 定位 和 IP 定位， 它们绕过了 JavaScript Geolocation API。

VanishMe 只能伪装 JavaScript API，无法改变：
• Wi-Fi 定位（扫描周围 Wi-Fi 热点）
• IP 地址定位（需要 VPN）
• 蓝牙定位
• GPS（手机）

如果你想让 Google Maps 显示假位置，需要：
1. 使用 VPN/代理改变 IP
2. 禁用 Wi-Fi（如果可能）
3. 使用隐身模式清除缓存
